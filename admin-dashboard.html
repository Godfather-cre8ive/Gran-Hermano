📝 How to Use This Dashboard

Use the sidebar navigation to edit any section of the GHA website. All changes are saved automatically to your browser's local storage.

  • Site Settings — Edit school name, contact info, address, social links
  • Hero Slides — Add, edit or remove homepage slideshow images
  • Admission — Update admission text, countdown date, exam dates
  • JAMB Scorers — Manage top scorer profiles and scores
  • School Sections — Edit Crèche, Nursery, Primary, Secondary content
  • Facilities — Add/edit/remove facility cards
  • News & Events — Publish, edit or delete news articles
  • Security — Change admin password

⚠ Changes are saved in browser localStorage. For production deployment, connect to a backend database.
